void appendOpenedFile ( char *  nomefile,
int  modo,
int  socket 
)

Aggiunge l' oggetto in input alla fine della lista linkata dei file aperti OpenedFile.

Se non esiste la radice mainFile la crea. Altrimenti scorre tutti i nodi fino ad arrivare all' elemento in coda, crea un nuovo nodo con i dati in input e lo appende alla fine.

Referenzia fileAlreadyOpen(), OpenedFile::fileName, getModoAperturaFromInt(), logM(), OpenedFile::modo, OpenedFile::next, openedFileLinkedList, prova(), e OpenedFile::socketId.

Referenziato da handleOpenCommand().

int fileAlreadyOpen ( char *  fileName,
modoApertura_t  modo,
int  socketId 
)

Controlla se un file con quel nome e' gia' stato aperto.

Parametri
fileNameIl nome del file da controllare
socketIdl' id della socket
Restituisce
1 se il file non e' aperto,
1 se il file e' gia' aperto, ed e' associato alla stessa socketId
1 se ha richiesto l'apertura in read
0 se il file e' gia' aperto in scrittura da altri

Referenzia FALSE, OpenedFile::fileName, logM(), OpenedFile::next, openedFileLinkedList, e TRUE.

Referenziato da appendOpenedFile().
